Spaceship Tower Defense with Cheat Slider Bar

To understand the basics of Javascript properly without using external libraries and dependencies, I built a Tower Defense Game of my own concept using only Vanilla JS. Random enemy ships have goal based path finding algorithm implemented due to which they find random new paths to destination cells. Our job is to stop these enemy ships by placing defense ships with various different powers. The powers of the ships have a cost associated. There is a cheat slider bar that helps make the game as easy or as difficult as the user wants. Click the button below to play as well as check the code of the application.
